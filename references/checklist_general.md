# 通用合同审查清单 (General Contract Review Checklist)

适用所有合同类型的通用审查维度与检查项。

## 一、主体资格审查

### 1.1 签约主体信息
- [ ] 甲方名称是否与营业执照完全一致
- [ ] 乙方名称是否与营业执照完全一致
- [ ] 统一社会信用代码是否填写正确
- [ ] 法定代表人/授权代表信息是否完整
- [ ] 联系人/地址/电话/邮箱是否完整

### 1.2 主体合法性
- [ ] 对方是否具备签约主体资格（是否已注销/吊销）
- [ ] 签约人是否获得有效授权（需提供授权委托书）
- [ ] 印章是否真实有效（核对公安备案编号）
- [ ] 特殊行业是否持有所需资质证书

### 1.3 履约能力
- [ ] 对方注册资本与合同金额是否匹配
- [ ] 是否存在失信被执行人记录
- [ ] 过往履约记录是否良好
- [ ] 是否存在重大诉讼/行政处罚

## 二、合同条款完整性

### 2.1 必备条款（民法典470条）
- [ ] 当事人的姓名或者名称和住所
- [ ] 标的（物品/服务/权利的具体描述）
- [ ] 数量（含计量单位和允许偏差）
- [ ] 质量（引用的国家标准/行业标准/企业标准）
- [ ] 价款或者报酬（含税/不含税明确）
- [ ] 履行期限、地点和方式
- [ ] 违约责任
- [ ] 解决争议的方法

### 2.2 常用补充条款
- [ ] 保密条款
- [ ] 知识产权条款
- [ ] 不可抗力条款
- [ ] 通知送达条款
- [ ] 合同变更/转让条款
- [ ] 合同解除条款
- [ ] 法律适用条款

## 三、条款表述精确性

### 3.1 语言精确度
- [ ] 无"大约""尽快""合理期间"等模糊表述
- [ ] 数字同时使用大写和小写
- [ ] 日期精确到年月日
- [ ] 金额明确币种和计量单位

### 3.2 定义清晰度
- [ ] 专业术语有明确定义
- [ ] 缩略语首次出现时有全称
- [ ] "包括但不限于"等表述含义明确
- [ ] 关联方/重大事项等范围有界定

## 四、违约责任

### 4.1 违约金
- [ ] 违约金是否合理（不超过实际损失30%，民法典585条）
- [ ] 违约金计算方式是否明确（如日千分之一）
- [ ] 是否存在惩罚性违约金（可能被法院调减）

### 4.2 违约情形
- [ ] 迟延履行违约责任是否约定
- [ ] 瑕疵履行违约责任是否约定
- [ ] 拒绝履行/部分履行责任是否约定
- [ ] 预期违约（履行期届满前拒绝履行）是否覆盖

### 4.3 免责条款
- [ ] 免责条款是否免除人身伤害责任（无效）
- [ ] 免责条款是否免除故意/重大过失责任（无效）
- [ ] 不可抗力条款是否覆盖常见情形

## 五、争议解决

### 5.1 管辖约定
- [ ] 是否约定诉讼或仲裁（不可同时约定，否则仲裁无效）
- [ ] 诉讼管辖法院是否明确（建议约定"合同签订地人民法院"）
- [ ] 是否违反专属管辖（不动产/港口作业/继承）
- [ ] 协议管辖是否属于与争议有实际联系的地点

### 5.2 仲裁条款
- [ ] 仲裁机构名称是否准确完整
- [ ] 仲裁地是否约定
- [ ] 仲裁规则是否指定
- [ ] 仲裁员人数和选任方式是否约定

## 六、法律合规性

### 6.1 强制性规定
- [ ] 是否违反法律/行政法规的强制性规定
- [ ] 是否损害社会公共利益
- [ ] 是否违反公序良俗
- [ ] 是否构成恶意串通损害他人利益

### 6.2 格式条款（民法典496-498条）
- [ ] 格式条款是否有提示说明义务
- [ ] 是否存在不合理免除/减轻己方责任
- [ ] 是否存在不合理加重对方责任
- [ ] 是否存在排除对方主要权利的条款

## 七、权利义务对等性

### 7.1 权利平衡
- [ ] 双方解除权是否对等
- [ ] 双方违约责任的严重程度是否匹配
- [ ] 是否存在单方任意变更权
- [ ] 是否存在单方任意解除权

### 7.2 弱势方保护
- [ ] 是否违反消费者权益保护
- [ ] 是否违反劳动者权益保护
- [ ] 是否存在显失公平的情形

## 八、履约可行性

### 8.1 交付/验收
- [ ] 交付标准是否具体可量化
- [ ] 验收程序和时间节点是否明确
- [ ] 验收不合格的处理方式是否约定
- [ ] 质保期和售后服务是否明确

### 8.2 付款安排
- [ ] 付款节点是否与履约进度匹配
- [ ] 付款方式是否安全（银行转账优先）
- [ ] 发票类型和开具时间是否明确
- [ ] 逾期付款的违约责任是否约定

## 九、附件管理

### 9.1 附件效力
- [ ] 附件是否注明"与正文具有同等法律效力"
- [ ] 附件是否编号并与主条款对应
- [ ] 附件内容是否与正文无矛盾
- [ ] 附件是否完整（无遗漏页）

## 风险等级判定

| 等级 | 符号 | 判定标准 | 行动建议 |
|------|------|----------|----------|
| 致命风险 | 🔴 | 可能导致合同无效或重大损失 | 必须修改，不可让步 |
| 重大风险 | 🟡 | 可能导致严重不利后果 | 原则上修改，可有限让步 |
| 一般风险 | 🟢 | 可协商改进的瑕疵 | 建议修改，可让步 |
