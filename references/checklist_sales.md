# 买卖合同审查清单

## 专项检查项

### 标的物条款
- [ ] 货物名称、规格、型号、品牌是否明确
- [ ] 是否附有清单作为附件
- [ ] 是否约定样品封存（凭样品买卖）

### 数量与计量
- [ ] 计量单位是否明确
- [ ] 是否约定合理损耗范围
- [ ] 溢短装条款是否存在（大宗货物）

### 质量条款
- [ ] 是否引用国家标准(GB)/行业标准
- [ ] 是否约定验收标准和验收方式
- [ ] 是否约定质保期及质保范围
- [ ] 是否约定质量异议期（民法典621条）

### 价款条款
- [ ] 单价和总价是否明确
- [ ] 是否注明含税/不含税
- [ ] 税率及发票类型是否约定
- [ ] 运费/保险费承担方是否明确

### 交付条款
- [ ] 交货地点是否明确到具体地址
- [ ] 交货时间是否具体到日期
- [ ] 运输方式和风险转移时间点
- [ ] 是否约定所有权保留（民法典641条）

### 验收条款
- [ ] 验收标准和验收方法是否明确
- [ ] 验收期限是否合理
- [ ] 验收不合格的处理方式
- [ ] 是否约定第三方检测机构

### 付款条款
- [ ] 预付款/定金比例是否合理（定金≤20%）
- [ ] 付款节点是否与交货/验收挂钩
- [ ] 逾期付款违约责任是否明确

### 特殊风险
- [ ] 一物二卖风险（不动产/特殊动产）
- [ ] 卖方无权处分风险
- [ ] 货物毁损灭失风险承担
