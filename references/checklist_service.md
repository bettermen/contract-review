# 服务合同审查清单

## 专项检查项

### 服务内容
- [ ] 服务内容/范围是否明确界定
- [ ] 是否约定具体服务标准/规格
- [ ] 是否附有服务说明书/方案
- [ ] 服务期限是否明确

### 服务质量
- [ ] 服务标准是否引用行业规范
- [ ] 验收标准是否可量化
- [ ] SLA（服务等级协议）是否约定
- [ ] 未达标时的补救措施是否明确

### 费用与支付
- [ ] 服务费计算方式是否明确（固定/按时/按效果）
- [ ] 预付款/分期付款安排是否合理
- [ ] 报销费用范围和审批流程是否明确
- [ ] 税费承担方是否明确

### 人员安排
- [ ] 服务人员资质要求是否明确
- [ ] 是否指定关键人员（不可更换）
- [ ] 人员变更的通知义务是否约定

### 成果归属
- [ ] 服务成果的知识产权归属是否明确
- [ ] 背景知识产权和前景知识产权是否区分
- [ ] 使用许可范围是否明确

### 保密
- [ ] 保密信息范围是否界定
- [ ] 保密期限是否明确（履行期间+终止后）
- [ ] 数据安全和隐私保护是否约定

### 变更管理
- [ ] 需求变更的处理流程是否明确
- [ ] 变更导致的费用调整机制是否约定
- [ ] 单方终止服务的条件是否合理

### 特殊风险
- [ ] 服务不达标的违约责任
- [ ] 数据丢失/泄露责任
- [ ] 依赖关键人员的履约风险
