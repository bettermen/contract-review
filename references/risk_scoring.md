# 风险评分方法论 (Risk Scoring Methodology)

标准化的合同风险评分算法，确保审查结果可量化、可比较、可追踪。

## 评分体系总览

### 综合评分公式

```
综合评分 = Σ(维度得分 × 维度权重) / Σ维度权重
```

### 评级映射

| 分数区间 | 评级 | 含义 | 建议 |
|----------|------|------|------|
| 90-100 | A+ | 优秀 | 可直接签署 |
| 80-89 | A | 良好 | 微调后签署 |
| 70-79 | B+ | 合格 | 小幅修改后签署 |
| 60-69 | B | 基本合格 | 建议修改部分条款 |
| 50-59 | C+ | 需关注 | 需修改后复审 |
| 40-49 | C | 风险较大 | 需大幅修改 |
| 30-39 | D | 高风险 | 不建议签署 |
| 0-29 | F | 极高风险 | 严禁签署 |

## 9维评分标准

### 维度1：主体资格 (权重: 15%)

| 检查项 | 扣分规则 |
|--------|----------|
| 主体名称缺失或不一致 | -20分 |
| 统一社会信用代码缺失 | -10分 |
| 法定代表人信息缺失 | -10分 |
| 无授权委托书（非法定代表人签署） | -15分 |
| 特殊行业无资质证书 | -25分 |
| 主体已注销/吊销 | -50分（直接致命） |

基准分: 100分，扣完为止，最低0分

### 维度2：条款完整性 (权重: 15%)

| 检查项 | 扣分规则 |
|--------|----------|
| 缺少标的条款 | -30分 |
| 缺少数量条款 | -15分 |
| 缺少质量标准 | -15分 |
| 缺少价款/报酬 | -20分 |
| 缺少履行期限 | -15分 |
| 缺少违约责任 | -25分 |
| 缺少争议解决 | -15分 |
| 缺少保密条款（需要时） | -10分 |

基准分: 100分

### 维度3：违约责任 (权重: 12%)

| 检查项 | 扣分规则 |
|--------|----------|
| 无违约金约定 | -20分 |
| 违约金过高（>损失30%） | -25分 |
| 违约金过低（无约束力） | -15分 |
| 违约情形不全面 | -15分 |
| 免责条款无效（免除人身伤害/故意过失） | -30分 |
| 单方免责条款 | -20分 |

基准分: 100分

### 维度4：争议解决 (权重: 10%)

| 检查项 | 扣分规则 |
|--------|----------|
| 同时约定诉讼和仲裁 | -40分（仲裁条款无效） |
| 管辖法院不明确 | -20分 |
| 违反专属管辖 | -35分 |
| 仲裁机构名称不准确 | -25分 |
| 管辖约定不利于己方 | -15分 |

基准分: 100分

### 维度5：法律合规 (权重: 15%)

| 检查项 | 扣分规则 |
|--------|----------|
| 违反强制性法律规定 | -50分（合同无效风险） |
| 损害社会公共利益 | -40分 |
| 违反公序良俗 | -35分 |
| 格式条款未履行提示义务 | -20分 |
| 不合理免除己方责任 | -25分 |
| 排除对方主要权利 | -30分 |

基准分: 100分

### 维度6：权利义务对等性 (权重: 10%)

| 检查项 | 扣分规则 |
|--------|----------|
| 双方解除权不对等 | -20分 |
| 违约责任严重不匹配 | -25分 |
| 单方任意变更权 | -20分 |
| 单方任意解除权 | -20分 |
| 存在霸王条款 | -30分 |
| 显失公平 | -25分 |

基准分: 100分

### 维度7：履约可行性 (权重: 10%)

| 检查项 | 扣分规则 |
|--------|----------|
| 交付标准不可量化 | -15分 |
| 验收程序不明确 | -15分 |
| 付款节点与履约不匹配 | -20分 |
| 质量标准不可操作 | -15分 |
| 履行期限不合理 | -10分 |

基准分: 100分

### 维度8：知识产权保护 (权重: 8%)

| 检查项 | 扣分规则 |
|--------|----------|
| 知识产权归属未约定 | -25分 |
| 保密条款不完整 | -15分 |
| 竞业限制不合理 | -20分 |
| 许可范围不明确 | -15分 |
| 后续改进成果归属不清 | -10分 |

基准分: 100分

### 维度9：格式规范性 (权重: 5%)

| 检查项 | 扣分规则 |
|--------|----------|
| 结构层次混乱 | -15分 |
| 存在模糊表述 | -10分 |
| 数字未大写 | -5分 |
| 日期不精确 | -5分 |
| 附件与正文矛盾 | -15分 |
| 附件不完整 | -10分 |

基准分: 100分

## 风险分级算法

### 风险等级判定

```python
def determine_risk_level(score, has_fatal_issue):
    if has_fatal_issue or score < 30:
        return "critical"  # 致命风险
    elif score < 50:
        return "major"     # 重大风险
    else:
        return "minor"     # 一般风险
```

### 致命风险触发条件（自动判定）

以下任一条件触发即标记为致命风险：
1. 主体已注销/吊销/无资质
2. 违反法律强制性规定
3. 合同标的违法
4. 免责条款免除人身伤害责任
5. 同时约定诉讼和仲裁
6. 违反专属管辖
7. 格式条款排除对方主要权利
8. 存在恶意串通条款
