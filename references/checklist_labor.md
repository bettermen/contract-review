# 劳动用工合同审查清单

## 专项检查项

### 合同期限
- [ ] 是否签订书面劳动合同（劳动合同法10条）
- [ ] 合同类型是否正确（固定/无固定/以完成一定任务为期限）
- [ ] 试用期是否合规（≤1月/2月/6月，劳动合同法19条）
- [ ] 无固定期限合同触发条件是否处理得当

### 薪酬条款
- [ ] 工资标准是否明确（不低于当地最低工资）
- [ ] 加班工资计算基数是否约定
- [ ] 绩效/奖金/津贴计算方式是否明确
- [ ] 工资支付日期是否明确

### 工时与休假
- [ ] 工时制度是否合规（标准/综合计算/不定时）
- [ ] 加班时长是否合规（每日≤3h，每月≤36h）
- [ ] 年休假/病假/婚假/产假安排是否合规

### 社保与福利
- [ ] 五险一金是否依法缴纳
- [ ] 补充商业保险是否约定
- [ ] 工伤处理流程是否明确

### 解除与终止
- [ ] 解除条件是否合法（劳动合同法39-41条）
- [ ] 经济补偿金计算标准是否约定
- [ ] 竞业限制条款是否合理（≤2年，需补偿）
- [ ] 离职交接和脱密期是否合理

### 竞业限制
- [ ] 竞业限制范围是否合理（非全员适用）
- [ ] 竞业限制期限是否≤2年
- [ ] 竞业限制补偿金是否约定（≥30%原工资）
- [ ] 违约金是否合理

### 保密义务
- [ ] 商业秘密范围是否界定清晰
- [ ] 保密期限是否合理
- [ ] 泄密违约责任是否明确

### 特殊风险
- [ ] 未签劳动合同双倍工资风险（劳动合同法82条）
- [ ] 违法解除劳动合同2倍赔偿风险
- [ ] 加班费纠纷风险
- [ ] 社保缴纳不合规风险
