# 建设工程合同审查清单

## 专项检查项

### 工程概况
- [ ] 工程名称/地点/规模是否明确
- [ ] 承包范围是否清晰界定
- [ ] 是否附有工程量清单
- [ ] 工程立项/规划/用地审批是否齐全

### 合同价款
- [ ] 合同价款模式（固定总价/可调价/成本加酬金）
- [ ] 价款调整机制是否明确
- [ ] 工程量变更的调价机制
- [ ] 预付款比例和扣回方式

### 工期管理
- [ ] 开工日期是否明确
- [ ] 竣工日期是否明确
- [ ] 工期延误的责任划分是否明确
- [ ] 工期顺延条件是否合理
- [ ] 误期损害赔偿（每日千分之一至三）

### 质量标准
- [ ] 质量标准（国标/行标/鲁班奖等）
- [ ] 隐蔽工程验收程序
- [ ] 竣工验收标准和程序
- [ ] 质保期和质保金比例（≤3%）

### 安全生产
- [ ] 安全生产责任是否明确
- [ ] 安全事故处理流程是否约定
- [ ] 工伤保险/意外保险是否约定

### 付款安排
- [ ] 工程进度款支付节点
- [ ] 竣工结算流程和时限
- [ ] 质保金返还条件
- [ ] 逾期付款违约责任

### 分包管理
- [ ] 是否允许分包
- [ ] 分包单位资质要求
- [ ] 分包管理责任
- [ ] 禁止转包

### 特殊风险
- [ ] 承包人资质不足
- [ ] 黑白合同风险
- [ ] 农民工工资支付风险
- [ ] 实际施工人诉讼风险
