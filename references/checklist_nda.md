# 保密协议审查清单

## 专项检查项

### 保密信息定义
- [ ] "保密信息"范围是否界定清晰（不可过于宽泛）
- [ ] 是否区分书面/口头/电子形式的保密信息
- [ ] 是否排除以下信息：公开信息/自行开发/第三方合法获取
- [ ] 保密信息标识要求是否合理

### 保密义务
- [ ] 保密义务主体是否明确（含关联方/员工）
- [ ] 保密期限是否合理（一般是合同期+N年）
- [ ] 使用限制是否明确（仅限约定目的使用）
- [ ] 复制/传输限制是否约定

### 信息归还/销毁
- [ ] 协议终止后是否要求归还或销毁
- [ ] 销毁证明要求是否约定
- [ ] 是否有留存例外（法律法规要求）
- [ ] 电子信息的删除是否约定

### 违约责任
- [ ] 违约赔偿范围是否明确
- [ ] 是否约定违约金（建议约定）
- [ ] 是否约定禁令救济（申请行为保全）
- [ ] 违约金是否合理（不宜过高）

### 竞业限制
- [ ] 是否包含不竞争条款（与竞业限制区分）
- [ ] 不招揽条款是否合理
- [ ] 限制期限和地域是否合理

### 特殊风险
- [ ] 单方保密协议的权利义务失衡
- [ ] 保密信息定义过于宽泛
- [ ] 缺少信息出口安全管控条款
- [ ] 缺少违约补救措施
