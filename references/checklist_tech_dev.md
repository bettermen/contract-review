# 技术开发合同审查清单

## 专项检查项

### 开发内容
- [ ] 技术目标是否具体明确
- [ ] 技术指标是否量化可验收
- [ ] 是否附有技术规格说明书
- [ ] 开发阶段和里程碑是否明确

### 知识产权
- [ ] 背景知识产权归属是否声明
- [ ] 前景知识产权归属是否明确（民法典865条）
- [ ] 专利申请权归属是否约定
- [ ] 软件著作权归属是否约定
- [ ] 后续改进成果的归属是否约定

### 开发周期
- [ ] 各阶段交付时间和交付物是否明确
- [ ] 迟延交付的违约责任
- [ ] 需求变更的流程和费用调整
- [ ] 不可抗力的工期顺延

### 验收标准
- [ ] 验收标准和测试方法是否明确
- [ ] 验收期限是否合理
- [ ] 验收不合格的补救措施
- [ ] 是否为"唯结果论"（可能导致无限次修改）

### 费用与支付
- [ ] 开发费用总额是否明确
- [ ] 付款节点是否与里程碑挂钩
- [ ] 差旅费等附加费用承担
- [ ] 需求变更导致的费用调整

### 保密
- [ ] 技术秘密的保护措施
- [ ] 源代码的保管和交付
- [ ] 开发人员的保密义务
- [ ] 反向工程的禁止

### 维护与支持
- [ ] 交付后的维护期和免费维护范围
- [ ] 技术支持和响应时间（SLA）
- [ ] Bug修复的时限
- [ ] 维保费用是否另行约定

### 特殊风险
- [ ] 技术成果未达到预期目标的风险分配
- [ ] 第三方知识产权的侵权风险
- [ ] 开源代码的合规使用
- [ ] 核心开发人员流动风险
- [ ] 技术交付物的"可用性"而非"完美性"标准
