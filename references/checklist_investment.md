# 投资协议审查清单

## 专项检查项

### 投资安排
- [ ] 投资金额/币种是否明确
- [ ] 估值（投前/投后）是否明确
- [ ] 股权比例计算是否准确
- [ ] 交割先决条件是否合理

### 股东权利
- [ ] 董事会/监事会席位安排
- [ ] 一票否决权范围是否合理
- [ ] 反稀释条款（加权平均/完全棘轮）
- [ ] 优先购买权/共同出售权
- [ ] 领售权（Drag-along）触发条件
- [ ] 随售权（Tag-along）安排

### 退出机制
- [ ] 回购权（Redemption）条件和价格是否合理
- [ ] IPO安排和合格上市标准
- [ ] 优先清算权（倍数/参与分配/不参与分配）
- [ ] 并购退出安排

### 陈述与保证
- [ ] 创始团队的陈述保证范围是否合理
- [ ] 赔偿上限是否约定（通常为投资额1-3倍）
- [ ] 赔偿时限是否合理（通常3-5年）
- [ ] 是否有赔偿下限（Basket/Deductible）

### 公司治理
- [ ] 信息权/检查权是否约定
- [ ] 年度预算审批流程
- [ ] 关联交易审批
- [ ] 竞业禁止条款

### 特殊条款
- [ ] 最惠国待遇（MFN）
- [ ] 估值调整机制（对赌）
- [ ] 创始人股份兑现（Vesting，通常4年+1年cliff）
- [ ] ESOP（员工期权池）安排

### 特殊风险
- [ ] 对赌条款的法律效力（与目标公司对赌可能无效）
- [ ] 外资准入限制（负面清单）
- [ ] VIE架构合规性
- [ ] 反垄断审查触发风险
