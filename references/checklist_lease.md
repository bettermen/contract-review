# 租赁合同审查清单

## 专项检查项

### 租赁物
- [ ] 租赁物描述是否清晰（地址/面积/用途）
- [ ] 出租人是否有权出租（产权证/授权委托）
- [ ] 租赁物是否存在抵押/查封
- [ ] 是否约定租赁物用途限制

### 租期与租金
- [ ] 租期是否明确（≤20年，民法典705条）
- [ ] 租金标准/支付周期/支付方式是否明确
- [ ] 租金递增机制是否合理
- [ ] 押金/保证金数额和退还条件是否明确
- [ ] 是否存在租金减免情形

### 维修与保养
- [ ] 维修责任分配是否明确（出租人一般维修义务）
- [ ] 大修/小修的责任边界
- [ ] 装修/改造是否需要出租人同意

### 转租
- [ ] 是否允许转租（民法典716条）
- [ ] 转租是否需要书面同意
- [ ] 转租收益分配是否约定

### 解除与返还
- [ ] 提前解除条件是否合理
- [ ] 装修补偿/损失赔偿是否约定
- [ ] 返还时的状态要求是否合理
- [ ] 逾期返还的占有使用费是否约定

### 优先权
- [ ] 优先购买权是否告知（民法典726条）
- [ ] 优先承租权是否约定

### 特殊风险
- [ ] 商用租赁的消防/环保合规
- [ ] 商铺租赁的经营许可
- [ ] 厂房租赁的安全生产资质
