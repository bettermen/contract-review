# Contract Review Skill for WorkBuddy

AI 合同智能审查助手 — WorkBuddy 本地技能，免费、离线可用。

## 特色

- 🎯 **9维雷达评分** — 主体资格/条款完整性/违约责任/争议解决/法律合规/权利对等/履约可行/知识产权/格式规范
- 🔥 **风险热力图** — 按合同章节可视化风险密度
- 📝 **条款对比修改** — 原文 vs 修改建议并列对比，可直接复制使用
- ⚖️ **法律依据引用** — 每个风险点引用《民法典》等具体法条
- 🎯 **谈判策略生成** — 不止发现问题，提供谈判话术和博弈角度
- 📋 **8类专项清单** — 买卖/劳动/租赁/服务/NDA/投资/建设工程/技术开发
- 🆓 **完全本地免费** — 无需订阅，离线可用
- 📊 **交互式HTML报告** — Chart.js雷达图 + 风险热力图 + 条款对比

## 安装

将 `contract-review/` 目录复制到 WorkBuddy 技能目录：

```
~/.workbuddy/skills/contract-review/
```

## 触发词

- 合同审查、审合同、合同风险、合同分析
- 帮我看看合同、合同审核、法律风险分析
- contract review

## 支持合同类型

| 类型 | 说明 |
|------|------|
| 买卖合同 | 购销/采购/供货 |
| 劳动用工合同 | 聘用/雇佣 |
| 租赁合同 | 房屋/设备租赁 |
| 服务合同 | 技术/咨询/委托服务 |
| 保密协议 | NDA/商业秘密 |
| 投资协议 | 股权/增资 |
| 建设工程合同 | 施工/承包 |
| 技术开发合同 | 软件开发/技术研发 |

## 免责声明

本技能输出的审查建议由AI生成，仅供参考，不构成法律意见。重大合同建议由专业律师最终审核。
